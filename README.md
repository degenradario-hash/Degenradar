# DegenRadar.io

> The All-in-One Crypto Intelligence Platform
> Scan. Detect. Profit.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features (MVP)

- **Memecoin Scanner** - Real-time token feed from DexScreener
- **Safety Score** - Basic safety analysis (liquidity, volume, age, buy/sell ratio)
- **Chain Filter** - Filter by Solana, Ethereum, Base, BSC, Arbitrum
- **Search** - Search tokens by name, symbol, or address
- **Auto-refresh** - Data updates every 30 seconds

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- DexScreener API (free)
- Vercel (hosting)
